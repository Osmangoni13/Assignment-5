Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents, representing the structure of a webpage as a hierarchical tree of objects. It allows programming languages like JavaScript to interact with and manipulate the content, structure, and style of a webpage dynamically.

How do you select an element from the DOM?

  • Use document.getElementById()
  • Use document.getElementsByClassName()
  • Use document.getElementsByTagName()
  • Use document.querySelector()
  • Use document.querySelectorAll()

What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events from its child elements. It is useful because it reduces the number of event listeners, improves performance, and allows handling dynamically added elements.

How do you manipulate an element's attributes?

  • Manipulate attributes: Use setAttribute(), getAttribute(), and removeAttribute().
  • Manipulate styles: Use the style property or modify classList.
